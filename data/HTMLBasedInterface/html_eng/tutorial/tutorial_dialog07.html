Creating a Solvent Excluded Surface

Next we want to visualise the peptides electrostatics. To do so, we will first create a Solvent Excluded Surface (SES), which we will colorize accordingly later.

To create a SES, highlight the System and open the context menu with the right mouse button. Select Create Representation ⇒ SES ⇒ by element. The resulting SES will show up in the "Representations" widget.
